GovCon Command Center

One-time setup — takes about 10 minutes total

STEP 1
Create a Notion Integration
Go to notion.so/my-integrations → New integration → Name it "GovCon Dashboard" → select your workspace → submit.
Copy the Internal Integration Secret token that appears. It starts with ntn_...
STEP 2
Share your 5 Notion databases with the integration
In Notion, open each of these databases. Click ••• → Connections → Connect to → GovCon Dashboard:

🎯 Opportunities Pipeline  ·  👥 Decision-Maker CRM  ·  ✉️ Outreach Log  ·  📁 Marketing Assets  ·  ✅ SBA Readiness
STEP 3
Deploy the Cloudflare Worker proxy (free, 2 minutes)
The Notion API blocks direct browser requests, so we use a tiny free proxy:

1. Go to workers.cloudflare.com → Create account (free) → Create Worker
2. Replace the default code with the contents of cloudflare-worker.js from your GovCon GitHub repo
3. Click Settings → Variables and Secrets → Add. Add TWO secrets:
   • NOTION_TOKEN = your Notion integration secret
   • SAM_API_KEY = optional, only needed if SAM.gov is re-enabled later
4. Deploy. Your Worker URL will look like: https://notion-proxy.yourname.workers.dev
STEP 4
Connect the dashboard
Enter your Cloudflare Worker URL below. The Notion token stays on the Worker — it never touches your browser or GitHub.
